ALGORITHMS IMPLEMENTED
======================

1. Needleman-Wunsch Global Alignment
   - Time: O(m×n) where m,n = sequence lengths
   - Space: O(m×n)
   - Scoring: BLOSUM62 matrix

2. UPGMA Hierarchical Clustering
   - Time: O(n³) where n = number of sequences
   - Build guide tree for progressive alignment
   - Average linkage distance metric

3. Progressive MSA
   - Align sequences following guide tree order
   - Combine pairwise alignments
   - Final refinement by iterative re-alignment

4. Pairwise Identity
   - Compare aligned sequences position by position
   - Ignore gaps in calculation
   - Range: 0.0 (completely different) to 1.0 (identical)

5. Conservation Analysis
   - Identify columns where all residues are identical
   - Exclude columns with any gaps
   - Report fraction and count

OUTPUT INTERPRETATION
====================

Pairwise Identity Matrix:
- Values 0.0-1.0 represent % identity between sequence pairs
- Diagonal = 1.0 (each sequence identical to itself)
- Off-diagonal = similarity between different sequences
- Symmetric: matrix[i,j] = matrix[j,i]

Average Pairwise Identity:
- Mean of all off-diagonal values
- >0.70: Well-conserved family
- 0.30-0.70: Divergent but homologous
- <0.30: Very distant sequences (may not be related)

Fraction Conserved:
- Number of positions where all residues match / total positions
- >0.50: Highly conserved alignment
- 0.10-0.50: Moderately conserved
- <0.10: Highly variable sequences
